# 여행 보조 앱 (Traveler App)

여행 상황에 특화된 날씨, 경비, 일정 관리 기능을 하나의 애플리케이션으로 통합한 실무 예제용 데모 프로젝트입니다.

---

## 1. 기획 목적

본 프로젝트는 **여행 상황에 특화된 날씨·경비·일정 관리 기능**을 하나의 애플리케이션으로 통합한 실무 예제용 데모를 제작하는 것을 목적으로 합니다.

실제 서비스 출시보다는 다음 역량을 보여주는 데 중점을 둡니다:
- 기능 설계 능력
- 사용자 흐름 이해
- 도메인 분리 능력

---

## 2. 핵심 컨셉

> **여행 전·중 사용 시나리오를 고려한 여행 보조 도구**

| 구분 | 내용 |
|------|------|
| 핵심 기능 | 일정, 날씨, 경비 관리 |
| 설계 방향 | 복잡한 기능보다 실무에서 자주 구현하는 기능의 집합 |

---

## 3. 주요 기능 구성

### 3-1. 날씨 확인 기능

#### 1) 내 위치 기반 날씨 확인
- 사용자의 현재 위치(GPS)를 기반으로 실시간 날씨 정보 제공
- 앱 실행 시 기본 화면으로 노출

**제공 정보:**
- 현재 기온
- 체감 온도
- 강수 여부
- 날씨 상태 아이콘 (맑음, 흐림, 비 등)

#### 2) 시간대별 날씨 확인
- 선택한 지역의 시간대별(3시간 또는 1시간 단위) 날씨 정보 제공
- 여행 일정 계획 시 활용 목적

**제공 정보:**
- 시간대별 기온 변화
- 강수 확률
- 날씨 상태 변화

#### 3) 타지역 날씨 확인 (국내·해외 도시)
- 사용자가 직접 도시명을 검색하여 날씨 확인
- 국내/해외 구분 없이 동일한 UI 제공
- 여행 예정지 사전 확인 용도

#### 4) 도시 저장 기능
- 자주 확인하는 도시를 즐겨찾기 형태로 저장
- 저장된 도시는 별도 목록에서 빠르게 접근 가능
- 여행지 여러 곳을 동시에 관리하는 시나리오 대응

---

### 3-2. 여행 경비 계산 및 기록 기능

#### 1) 여행 도시 및 통화 선택
- 여행할 도시 선택
- 통화 선택 기능 제공
- 기본값: 대한민국 / KRW
- 해외 여행 시 자동으로 외화 기준 계산 가능

#### 2) 환율 자동 계산 기능
- 외국 통화 선택 시 환율 자동 적용
- 환율 계산 결과 표시 방식 선택:
  - 원화 기준으로 보기
  - 현지 통화 기준으로 보기
- on/off 옵션 제공하여 데모용 인터랙션 강조

#### 3) 여행용 계산기
- 간단한 금액 계산 기능
- 환율 선택 상태에 따라 자동 반영
- 예: 식비, 교통비, 쇼핑 금액 즉시 계산

#### 4) 지출 경비 기록
- 항목별 지출 내역 기록
- 여행 중 실제 지출 관리 시나리오 구현

**기록 항목:**
- 날짜
- 사용 금액
- 통화
- 간단한 메모

---

### 3-3. 여행 일정 기록 기능

#### 1) 캘린더 기반 여행 일정 저장
- 캘린더 UI를 활용한 여행 일정 기록
- 일정과 날씨 확인을 함께 사용하는 흐름 고려

**일정 정보:**
- 날짜
- 방문 도시
- 간단한 일정 메모
